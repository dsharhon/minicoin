# minicoin
An educational cryptocurrency.

By Dylan Sharhon.

Based on [Naivecoin](https://lhartikk.github.io/) by [Lauri Hartikka](https://github.com/lhartikk).

## Features
* P2P relaying
* POW mining
* UTXO transactions
* Mempool

## Running
    npm install
    node src/main.js
